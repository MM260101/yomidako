# 「よみだこ」要件定義

本ページに、Java/SpringをベースとしたQR打刻システム **「よみだこ」** の制作資料を記載します。<br>
企画書、要件定義書等の資料はGoogleドライブを使用し共有しています。また、ER図はサイズが膨大なため、Mermaidを用いた記述を共有をするものとします。
